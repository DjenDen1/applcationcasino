Требования для запуска приложения:


  1. Скачать ZIP-файл и после извлечения файлов достать папку "casinoloot2".

  2.  По файловому пути casinoloot2\build\5_15_17_mingw64_static-Debug\debug найти исполняемый файл "casinoloot.exe" (для дальнейшего запуска).

  3. Для избежения проблем с запуском программы, рекомендуется установить приложение "пакет совместимости"  -  https://apps.microsoft.com/detail/9NQPSL29BFFF?hl=ru-ru&gl=RU&ocid=pdpshare .

  4.  (Последний пункт необязателен) скопировать исполняемый файл и перенести в удобное место.


Изменения в данном коммите:

  1. Реализовао меню настроек -  будущее  окно  в которе будет объединять другие игры. На данный момент меню настроек может выполнять только одну опцию - сделать ставку и показывать число ставки.

  2.  Полноценная игра "Рулетка" на данный момент явлется достаточно простой игрой:

                  . В игре пока что не учитываются правила настоящей рулеткию.

                  . В окне игры можно сделать ставку. Ставка работает аддитивно - если выигрыш,  прибавляется на ставку, сделанную пользователем, в случае проигрыша уменьшается на ставку, заданную пользователем.

                  .  Имеет простое оформление.
  
  3.  Полноценная игры "Автомат" на данный момент является простой игрой:

                  . Имеет простое и кривое оформление.

                  . В окне игры можно сделать ставку. Ставка работает аддитивно - если выигрыш, прибавляется на ставку, сделанную пользователем, в случае проигрыша уменьшается на ставку, заданную пользователем.

                  . Сам автомат не имеет никагого дизайна.
 
 4. Во всех действущих играх , на момет данного коммита, просмотр количества фишек, которое изменяется посредством   ставок на игры (Рулетка, Автомат), основан на мини дисплее,

    для того чтобы видеть актуальную информацию о балансе фишек необходимо нажимать на кнопку "Баланс фишек".
